# 👋 Hola, soy Godo Sanchez

Desarrollador de sistemas embebidos enfocado en firmware, electrónica aplicada, adquisición de datos, control y comunicación industrial. Me interesa crear soluciones donde los microcontroladores, sensores y protocolos de comunicación trabajen juntos para resolver problemas reales en campo.

---

## 🧐 More About Me

🔭 Trabajo en proyectos de firmware, IoT industrial, sensores, CAN Bus, LoRaWAN, BLE y sistemas embebidos.  
🌱 Actualmente profundizo en STM32, ESP32, Linux embebido, LVGL, Qt, DSP y arquitecturas de firmware escalables.  
⚙️ Me interesa el desarrollo de soluciones robustas para monitoreo, automatización, control y adquisición de datos.  
🧠 Áreas de interés: ARM, PIC, dsPIC, AVR, procesamiento digital de señales y control moderno.  

---

## 💻 Sample Projects

- Firmware para STM32 usando HAL, CubeIDE y comunicación CAN Bus.
- Aplicaciones con ESP32, FreeRTOS, sensores y comunicación inalámbrica.
- Interfaces gráficas embebidas con LVGL y Qt.
- Sistemas de monitoreo industrial con sensores, gateway y dashboard.
- Pruebas con LoRaWAN, BLE, WiFi, UART, SPI, I2C y CAN.
- Automatización de compilación y despliegue de firmware con Git, Docker y Jenkins.
